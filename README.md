# Resume Website - Dhruv Agrawal

A responsive personal resume website with downloadable PDF functionality.

## Features
- Fully responsive design (mobile, tablet, desktop)
- Downloadable resume (print to PDF)
- Clean and professional layout
- Easy to customize

## How to Deploy to GitHub Pages

1. Create a new repository on GitHub named `yourusername.github.io`
2. Initialize git and push the code:
```bash
cd Documents/resume-website
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/yourusername/yourusername.github.io.git
git push -u origin main
```

3. Your website will be live at: `https://yourusername.github.io`

## How to Customize

Edit `index.html` to update:
- Personal information (name, email, links)
- Work experience
- Education details
- Skills
- Projects

Edit `style.css` to change:
- Colors
- Fonts
- Layout
- Spacing

## Local Testing

Simply open `index.html` in your browser to preview the website.
